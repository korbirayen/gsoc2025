Django

Slogan: Web framework for perfectionists with deadlines

Tech Stack: python, django

Topic: web, python

Description: Web framework for perfectionists with deadlines Django is a high-level Python Web framework originally developed at the Lawrence-Journal World. Django was designed to handle two challenges: the intensive deadlines of a newsroom and the stringent requirements of the experienced Web developers who wrote it. It lets you build high-performing, elegant Web applications quickly.
