KubeVirt

Slogan: Building a virtualization API for Kubernetes

Tech Stack: golang, grpc

Topic: virtualization, containers, kubernetes

Description: Building a virtualization API for Kubernetes KubeVirt is a virtual machine management extension for Kubenetes, allowing you to create and manage virtualised workloads natively alongside container workloads. It does this by adding virtualization resources through the Kubernetes Custom Resource Definition API. KubeVirt is a Cloud Native Computing Project (CNCF) incubating project.
